# Town-Trek-- online-travel-guidance
TownTrek is a visually engaging travel experience website designed to highlight the natural beauty, cultural richness, and adventurous destinations of Assam. It combines an immersive UI with smooth navigation and interactive trip booking features.

*Key Website Features*
• Full-screen animated carousel with overlay text introducing the theme of exploration.
• Floating responsive navbar with internal links for seamless section navigation.
• Destination showcase with animated cards that expand on hover to reveal full
descriptions.
• Smooth scrolling to About, Adventures, and Contact sections.

**Trip Booking System**
• Users can choose trips from four destinations ̶ Kamakhya Temple, Kaziranga National
Park, Majuli Island,
and Haflong Hills.
• Each trip contains predefined duration and cost values.
• Users can add multiple trips and instantly view a summary with total days and total cost.
• Fully client-side JavaScript implementation ensures instant feedback.

**Design & User Interface**
• Large hero visuals and minimalistic content layering.
• Fully responsive layout suitable for mobile, tablet, and desktop.
• Card hover animations improve user engagement and enhance readability.
